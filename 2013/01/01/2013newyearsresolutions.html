2013е№ҙгҒ®жҠұиІ 

ж–°е№ҙгҒӮгҒ‘гҒҫгҒ—гҒҰгҒҠгӮҒгҒ§гҒЁгҒҶгҒ”гҒ–гҒ„гҒҫгҒҷгҖӮ

гҒқгӮҢгҒ§гҒҜ е№ҙжң«гҒ®жҢҜгӮҠиҝ”гӮҠ гҒ®з¶ҡгҒҚгӮ’жӣёгҒҚгҒҫгҒҷгҖӮ

гҒЎгҒӘгҒҝгҒ«ж°—гҒ«гҒӘгӮӢд»Ҡе№ҙгҒ®гҒҠгҒҝгҒҸгҒҳгҒ®зөҗжһңгҒҜдёӯеҗүгҒ§гҒ—гҒҹгҖӮ

% python -c 'import random; print random.choice(["еӨ§еҗү","дёӯеҗү","е°Ҹеҗү","еҮ¶","еӨ§еҮ¶"])'
дёӯеҗү

1. иҮӘеҲҶгҒҢгӮігғҹгғҘгғӢгғҶгӮЈеҶ…еӨ–гҒ§гҒ©гҒҶгҒӘгӮҠгҒҹгҒ„гҒ®гҒӢгӮ’иҖғгҒҲгҖҒе®ҹиЎҢгҒ—з¶ҡгҒ‘гӮӢ

гҒҫгҒ дҪ•гӮ’гӮ„гӮӢгҒ®гҒӢе…·дҪ“зҡ„гҒ«гҒҜжұәгҒҫгҒЈгҒҰгҒ„гҒҫгҒӣгӮ“гҖӮгҒ“гӮҢгҒҜе№ҙй–“йҖҡгҒ—гҒҰиҖғгҒҲгҖҒйғҪеәҰе®ҹиЎҢгҒ—з¶ҡгҒ‘гҒӘгҒ„гҒЁгҒ„гҒ‘гҒӘгҒ„гҒӘгҒЁжҖқгҒЈгҒҰгҒ„гӮӢгҒЁгҒ“гӮҚгҒ§гҒҷгҖӮиҖғгҒҲгӮӢгғқгӮӨгғігғҲгҒЁгҒ—гҒҰгҒҜгҖҒгҖҢеҠӘеҠӣгҒҷгӮӢгғҷгӮҜгғҲгғ«гҒҢзӣ®жҢҮгҒҷгӮҙгғјгғ«гҒЁдёҖиҮҙгҒ—гҒҰгҒ„гҒӘгҒ„гҒЁж„Ҹе‘ігҒҢз„ЎгҒ„гҖӮгҖҚгҒЁгҒ„гҒҶгҒ“гҒЁгҒ§гҒҷгҒӢгҒӯгҖӮ

2. pull request гғҮгғ“гғҘгғј

еҚҳгҒ«OSSгӮігғҹгғҘгғӢгғҶгӮЈгҒ«дҪ•гҒӢиІўзҢ®гҒҷгӮӢгҒ«гҒҜдҪ•гҒҢгҒӮгӮӢгҒӢгҒӘгҒЁжҖқгҒЈгҒҹжҷӮгҒ«гҒ“гӮҢгҒӢгҒӘгҒЁжҖқгҒ„гҒҫгҒ—гҒҹгҖӮгғҰгғјгӮ¶гғјгҒЁгҒ—гҒҰдҪҝз”ЁгҒ—гҒҹгӮҠгҖҒе®ЈдјқгҖҒеёғж•ҷгҒ—гҒҹгӮҠгҖҒе®ҹйҡӣгҒ«й–ӢзҷәгҒ—гҒҹгӮҠгҒЁиІўзҢ®гҒ®д»•ж–№гҒҜж§ҳгҖ…гҒ гҒЁжҖқгҒ„гҒҫгҒҷгҒҢгҖҒд»Ҡе№ҙгҒҜгӮігғҹгғғгғҲгҒ—гҒҰгҒ„гҒ“гҒҶгҒӢгҒӘгҒЁгҖӮгҒҫгҒҡгҒҜissueзҷ»йҢІгҒӢгӮүгҒ«гҒӘгӮӢгҒЁжҖқгҒ„гҒҫгҒҷгҒҢгҖҒгҒјгҒЎгҒјгҒЎгӮ„гҒЈгҒҰгҒҝгҒҹгҒ„гҒЁжҖқгҒ„гҒҫгҒҷгҖӮ

3. гӮӨгғҷгғігғҲгӮ№гӮҝгғғгғ•гғҮгғ“гғҘгғј

2012е№ҙгҒҜж§ҳгҖ…гҒӘгӮӨгғҷгғігғҲгҒ«еҸӮеҠ гҒ—гҒҫгҒ—гҒҹгҖӮ2013е№ҙгҒҜ PyCon APAC гӮӮй–ӢгҒӢгӮҢгӮӢгҒ“гҒЁгҒ§гҒҷгҒ—гҖҒгӮ№гӮҝгғғгғ•гҒЁгҒ—гҒҰгӮӨгғҷгғігғҲгҒ«й–ўгӮҸгӮҠгҒҹгҒ„гҒЁжҖқгҒ„гҒҫгҒҷгҖӮгҒ—гҒӢгҒ—еӢӨеӢҷең°гҒҢйҒ гҒ„гҒЁгҒ„гҒҶжңҖеӨ§гҒ®жҮёеҝөзӮ№гҒҢгҒӮгӮҠгҒҫгҒҷгҖӮйҒ йҡ”гҒ§еҸӮеҠ гҒ§гҒҚгӮӢеҪўгӮ’иҰӢгҒӨгҒ‘гӮӢгҖӮеҪ“ж—ҘгӮ№гӮҝгғғгғ•гҒЁгҒ—гҒҰжҗәгӮҸгӮӢгҖӮгҒӘгҒ©гҒЁгҒ„гҒҶеҪўгӮ’иҖғгҒҲгҒ—гҒЈгҒӢгӮҠгҒЁжҗәгӮҸгӮҢгӮӢгӮҲгҒҶгҒ«гҒ—гҒҰеӢ•гҒ„гҒҰгҒ„гҒҚгҒҫгҒҷгҖӮ

2013е№ҙвҖҰвҖҰ

д»Ҡе№ҙгҒҜе·іе№ҙгҖҒгҒЁгҒ„гҒҶгҒ“гҒЁгҒҜPythonгҒ®е№ҙгҒ§гҒҷгҒӯгҖӮе®ҹгҒҜз§ҒгҒҜе·іе№ҙгҒ§е№ҙз”·гҒ«гҒӘгҒЈгҒҰгҒҠгӮҠгҒҫгҒҷгҖӮд»Ҡе№ҙгҒҜжҳҜйқһйЈӣиәҚгҒ®е№ҙгҒ«гҒ—гҒҹгҒ„гҒ§гҒҷгҒӯгҖӮгҒҝгҒӘгҒ•гӮ“жң¬е№ҙеәҰгӮӮгӮҲгӮҚгҒ—гҒҸгҒҠйЎҳгҒ„иҮҙгҒ—гҒҫгҒҷ:D
